Pomodoro Insertion Test

This page provides instructions for testing whether pomodoro records are properly inserted into the database when a focus timer completes.

Note: Make sure both the React frontend and Node.js server are running before proceeding with the tests.

Option 1: Use the Test Tab

The application has a built-in test tab with testing components:

  1. Start the application (cd pomodoro-timer && npm start)
  2. Start the server (cd server && npm run dev)
  3. Log in to your account
  4. Click on the "Test" tab in the navigation bar
  5. Scroll down to find the "Pomodoro Insertion Test" and "Direct Pomodoro API Test" sections

Option 2: Use the Standalone Test Page

If you can't access the Test tab, you can use our standalone test page:

  1. Make sure the application is running
  2. Click the button below to open the standalone test page
  3. Follow the instructions on the test page
Open Standalone Test Page

What to Look For

Both tests will show:

  • A success message if a new pomodoro record was inserted
  • Details about the inserted record, including project ID and task ID
  • The count of records before and after the test
Troubleshooting: If you encounter any issues, make sure:
  • You are logged in to your account
  • Both the frontend and server are running
  • You have selected a project before running the tests
